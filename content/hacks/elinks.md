<ESC> opens menu item
h - History
/ ? - Search
\ - Toggle HTML/Plain
* - Toggle images
. - Toggle link numbering
% - Toggle document colors
W - Toggle wrap text
> - Next tab
< - Previous tab
c - Close tab
Ctrl-R - Reload
Right - Go to link
Right + Enter - Reload and go to link
d - download link
A - Add bookmark
