Gmail tips and tricks

Posted by Ravikiran K.S. on January 1, 2006

Gmail Tips

label:abc – List all messages belonging to a label is:unread – List all messages unread from: – List all messages from person with subject: – List all messages with given word in subject line

To get help on some query, just enter query type. For ex. “is:” This will provide suggestions.

Gmail Shortcuts

c - compose / - search e - archive n/p - next/previous message - open message x - select message # - delete message r/a/f - reply, reply-all, forward l - label z - undo

No importance for dot

If your email address is GeorgeBush@gmail.com, then everything sent to any of the following email addresses will be forwarded to your primary email.

GeorgeBush@gmail.com OR GeorgeBush@googlemail.com G.eorgeBush@gmail.com OR G.eorgeBush@googlemail.com Ge.orgeBush@gmail.com OR Ge.orgeBush@googlemail.com …… GeorgeBus.h@gmail.com OR GeorgeBus.h@googlemail.com
